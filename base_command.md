#### ✓ Команды терминала PostgreSQL

▪ Войти в терминал. Запустить оболочку PostgreSQL

    sudo -u postgres psql

▪ \l - проверить список доступных баз данных
  
▪ \c - подключиться к базе данных

▪ \d - просмотр таблиц


▪ \q - выход из базы данных
